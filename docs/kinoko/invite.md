## 什么是领养菌菌？
对于非官方QQbot来说，大致的结构是`本地的QQ框架 → Bot服务器`。其中的“QQ框架”和“Bot服务器”其实都可以放在本地，就像菌菌之前的那样，使用一个QQ号供大家使用。

现改变领养方式，将“QQ框架”和“Bot服务器”分开，由大家自己建立“QQ框架”的部分，然后连接到菌菌的服务器。

简单来说，就是**用自己的账号作为菌菌使用。**
是否邀请进入其它群、是否接受加好友等之前菌菌会限制的部分，现在都由账号持有者决定。

## 领养前必须知道的事情

::: danger
菌菌尽可能保证提供的服务绿色安全，但仅是将QQ号用作Bot这一行为本身，就已经违反了QQ的用户协议，不论用途。

**你用作领养菌菌的QQ号可能会因此掉线、限制社交服务或封禁。解除封禁需要给QQ上传身份证号并人脸识别认证，尽管如此也有无法解封的可能性。**

要领养菌菌，就默认你已经阅读了上述内容。由于部署Bot而导致QQ号异常的后果，菌菌不予承担。
:::

## 领养要求和其它事项
::: warning 领养要求
- 你需要一定的电脑操作知识
- 你需要一台本地的电脑、本地的NAS、云服务器等可以联网并运行程序的机器，最好可以使用docker
- 你需要[加入菌菌群](../kinoko/group)或者直接[联系开发者](../kinoko/group.md#联系开发者)本人，提供用作Bot的QQ号，用于加入白名单
- **在提供QQ号后，开发者会发送具体的部署方法**
:::

::: tip 其它事项
- 搭建“QQ框架”的那台电脑最好就在自己所在地（可能会检测异地登录
- 菌菌有义务保证用户的隐私，不会保存、转发用户的聊天信息
- 领养菌菌后，对于是否被邀请进入其它群、是否同意某人的好友操作完全由QQ号持有者决定，无需说明
:::
