### 在群里用官方Bot版菌菌的话需要@才可以触发消息

> 对于下方的蓝色粗体字，可以直接点击进入对应功能的详细文档。
> 
> [点击此处](../index.md)回到文档首页。
___

### 查歌
**指令示例：** `查歌 草` 或 `/查歌 草`
- 按关键词[查歌](../taiko/search.md)
- `/查歌` 是给官方bot用的，和不加 `/` 时完全一样
- 回复带有id的歌曲列表

___

### 查谱面
**指令示例：** `鬼id123`
- 用难度和id[查谱面](../taiko/search.md#再通过id查询谱面)
- 官方菌菌使用时建议长按头像来@

___

### 别名查歌
**指令示例：** `草是什么歌` 或 `/别名查歌 草`
- 按照[别名查](../taiko/alia-search.md)谱面
- `/别名查歌` 是方便官方bot避免@菌菌用的 (只是把`是什么歌`前面的关键词移到`/别名查歌`后面而已)

___

### 歌曲位置
**指令示例：** `草歌在哪` 或 `id123歌在哪` 或 `位置id123`
- 用别名查[歌曲位置](../taiko/song-location.md)
- 也可以用id来查

___

### 段位查询
**指令示例：**
- `/段位`
- `/段位 十段`
- `/段位 2025秋`
- `/段位 2025秋 十段`

**说明(从上至下依次是)：**
- 查看 "[/段位](../taiko/dani.md)" 功能的使用方式
- 查历代段位
- 查版本内所有段位
- 查段位详情

___

### 绑定广场
**指令示例：** `/绑定广场 12345678`
- [绑定](../taiko/donderhiroba.md)自己的鼓众广场账号
- 按提示操作

___

### 我的小咚
**指令示例：** `/我的小咚`
- [看小咚](../taiko/donderhiroba.md#绑定了之后可以做什么)和成绩面板
- 当然，需要先绑定

___

### Rating查询
**指令示例：**
- `/rt`
- `/rt 10星`
- `/rt 百花缭乱`
- `/rt 季曲系列`
- `/rt 10.4`

**说明(从上至下依次是)：**
- 查[Rating](../taiko/rating.md)分表
- 按星级来查定数表
- 按照别名查歌的方式，查单曲rt分
- 当一个别名对应多首歌曲时，可以查这些所有歌曲的rt分
- 别名里也填了菌菌难度定数，可以直接这样查（其实本质还是按别名查。并非Rating定数）
